# terraform-aws-cockroach

Terraform module for building CockroachDB cluster infrastructure on AWS.

# Usage

Define variables in `.vars`:

    aws_access_key = ""
    aws_secret_key = ""
    aws_region = "eu-north-1"
    owner = ""
    cluster_user_cidr = ["/32"]
    loaders = 0
    monitor = 0

Execute:

    $ terraform apply -var-file=.vars

Destroy:

    $ terraform destroy
